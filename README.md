📊 Análisis de Ventas y Rendimiento de las Tiendas de Alura Store
📖 Descripción del proyecto

Este proyecto tiene como objetivo analizar el rendimiento de las cuatro tiendas de Alura Store con el fin de ayudar al Sr. Juan a decidir cuál de ellas debería vender para invertir en un nuevo emprendimiento.

Para tomar esta decisión se realizó un análisis exploratorio de datos utilizando Python y Google Colab, evaluando diferentes indicadores como:

Facturación total por tienda

Categorías de productos más vendidas

Calificación promedio de los clientes

Productos más y menos vendidos

Costo promedio de envío

A partir de estos análisis se generaron visualizaciones y una recomendación final basada en los datos.

📂 Datos utilizados

El proyecto utiliza 4 conjuntos de datos correspondientes a cada tienda:

tienda_1.csv

tienda_2.csv

tienda_3.csv

tienda_4.csv

Cada dataset contiene información como:

Producto

Categoría del producto

Precio

Costo de envío

Lugar de compra

Calificación del cliente

🛠 Tecnologías utilizadas

Python

Pandas → Manipulación y análisis de datos

Matplotlib → Visualización de datos
